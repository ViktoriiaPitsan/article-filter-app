## üöÄ Features

- **Real-time Data**: Integrated with The Guardian API to fetch the latest 20 articles in the "Environment" section.
- **Live Search & Filtering**: Users can filter articles by keywords in the title or summary.
- **Keyword Highlighting**: Matches in titles and descriptions are highlighted in real-time as you type.
- **Fully Responsive**: Mobile-first design using CSS Grid and Flexbox to ensure zero horizontal scrolling on all devices.
- **Interactive UI**: Smooth hover effects and transitions for better user engagement.
- **Detailed Views**: Dynamic routing to individual article pages with full content rendering.

## üì¶ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ViktoriiaPitsan/article-filter-app.git
   ```

2. **Enter the directory**:
   ```bash
   cd article-filter-app
   ```
3. **Install dependencies:**:
   ```bash
   npm install
   ```

4. **Set up API Key**:
   The project uses a demo key for The Guardian API. You can replace it in useArticles.ts and ArticlePage.tsx.

5. **Run the app**:
   ```bash
   npm run dev
   ```
